## About

This repository contains my personal study materials for **Intermediate Macroeconomics**.

The materials come from two main sources:

* **NCCU (National Chengchi University)** — lecture content and class notes from the Intermediate Macroeconomics course
* **NTU (National Taiwan University)** — notes taken while auditing a related macroeconomics course

The repository also includes the professor's **in-class slides (PPTs)**, my own **graduate entrance exam preparation notes**, and **supplementary materials** collected while studying.

These materials may be helpful if you are:

* Taking Intermediate Macroeconomics
* Reviewing macroeconomic theory
* Preparing for exams
* Preparing for economics-related graduate entrance exams
* Looking for additional explanations or class slides

Please note that this is a personal note collection, not an official course website.

## How to Use These Notes

These notes are intended to be used as a **study aid**.

A recommended way to use this repository is:

* Start with `PPTs/` to follow the original in-class material
* Use `lecture-notes/NCCU-notes/` for exam-focused explanations
* Use `lecture-notes/NTU-audit-notes/` for an alternative perspective on the same topics
* Refer to `supplement/` for additional readings and clarifications
* Use `grad-prep-notes/` for graduate entrance exam preparation

These notes work best when used alongside lectures, textbooks, and problem-solving practice.

## Notes and Limitations

These are **student-written notes**, so they may contain mistakes, incomplete explanations, or personal interpretations.

Since the notes come from two different courses (NCCU and NTU), explanations, notation, and emphasis may vary between the `NCCU-notes/` and `NTU-audit-notes/` folders.

If you find any errors, unclear explanations, or formatting problems, feedback is welcome.
